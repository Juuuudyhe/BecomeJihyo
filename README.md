# Juuuudy's Training

個人健身追蹤 PWA，記錄訓練重量、感受度、飲食、月經週期。

## 功能

- 💪 四天分化訓練菜單（胸/腿/肩/背）
- 📊 訓練重量與進步圖表
- 🍱 飲食拍照記錄
- 🌙 月經週期追蹤與訓練建議
- 📤 一鍵匯出本週紀錄（給 AI 分析）
- 💾 全部資料匯出/匯入（換手機用）

## 使用方式

直接訪問：`https://juuuudyhe.github.io/training/`

iPhone 加到主畫面：
1. Safari 開啟網址
2. 分享 → 加入主畫面
3. 名稱清空，加入

## 技術

- 純前端 HTML/CSS/JS，無框架
- 資料存 localStorage（同裝置同瀏覽器）
- 圖示內嵌（無外部依賴）
